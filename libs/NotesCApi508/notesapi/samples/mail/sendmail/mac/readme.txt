SENDMAIL
--------
sendmail -  Send a Notes Mail Message using the Notes Mail Gateway API 
            functions.


ENVIRONMENTS
------------
MacOS (8.x) Power PC


FILES
-----
readme.txt  - This file.

sendmail.c  - Main program.
macui.c     - Dialog processing routines for Macintosh.
sendmail.h  - Program defines, control IDs, and function prototypes.
sendmerr.h  - Defines of all string table resource IDs.

sendmail.r  - Resource description file for Macintosh containing
              dialog box and string table.


RUNNING sendmail
----------------
To run this program on the Macintosh, double click the program icon.

SENDMAIL will display a Send Notes Mail dialog box on the screen.
Complete the dialog box and click SEND to send a Notes mail memo.
